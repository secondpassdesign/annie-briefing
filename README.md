# annie-briefing

Public Flash Briefing feed for Alexa, generated by Annie (Vellum assistant).

The file `feed.json` is rewritten each morning after the daily briefing runs. Alexa reads the latest `mainText` when you say:

> "Alexa, what's my briefing"

## Layout

- `feed.json` — Amazon Flash Briefing format (uid, updateDate, titleText, mainText)
- Public URL once Pages is enabled: `https://secondpassdesign.github.io/annie-briefing/feed.json`

## Update cadence

- Refreshed by Annie at ~7:35 AM PT weekdays and ~8:35 AM PT Saturdays, immediately after each daily briefing run.
- Each update is a single commit; history preserved.
